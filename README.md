# AI Course Recommendation Engine 🚀

## Overview

This project is a full-stack AI-powered course recommendation system that suggests personalized learning paths based on user interests, skills, and goals.

## Tech Stack

* Frontend: React
* Backend: Spring Boot (Java)
* AI Layer: (Planned: LLM / LangChain)
* Database: (Planned: MySQL)
* Deployment: AWS (Planned)

## Features

* User input-based recommendations
* REST API architecture
* Scalable full-stack design

## Project Structure

```
backend/   → Spring Boot API  
frontend/  → React UI  
```

## How to Run

### Backend

```
cd backend
mvn spring-boot:run
```

### Frontend

```
cd frontend
npm install
npm start
```
